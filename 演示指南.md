# 曹操民宿 · 自动记账提成系统 演示指南

## 快速演示（三步走）

### 第一步：运行系统
在 VS Code 终端（或 CMD）中执行：
```bash
cd d:\Projects\APUS
python caocao_minsu_demo/caocao_demo.py
```

## 演示话术（对别人讲）

### 开场（10秒）
> "这就是曹操那个案例的Demo版——用微信群+自动脚本解决员工虚报收入的问题。"

### 操作（30秒）
> "运行一下，看效果：`python caocao_minsu_demo/caocao_demo.py`"

### 讲解要点（2分钟）
1. **控制台直接展示** - 42条消息全部解析成功
2. **自动算提成** - 阿里23单8100迪拉姆、哈桑13单4050迪拉姆
3. **异常检测亮红灯** - 发现阿里6月22日虚报800迪拉姆（标准价才300）
4. **月度报告自动生成** - 打开结算报告_2026年6月.md 给看

### 核心观点（30秒）
> "全程就两步：员工在群里发消息 → 系统自动跑完。老板就算在巴塞罗那，也能一清二楚。这个思路可以复制到任何行业。"

## 进阶演示

### 如果时间充裕，打开结算报告
```bash
code caocao_minsu_demo/结算报告_2026年6月.md
```
在 VS Code 中预览 Markdown 格式的正式报告，看起来像一份正式的商务文档。

### 如果问"真实场景怎么接入"
把脚本中的 `get_sample_messages()` 函数里的模拟数据，替换成微信群导出的真实聊天记录即可，解析逻辑完全一样。
